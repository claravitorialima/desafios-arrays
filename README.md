# desafios-arrays

## 🎯 Descrição da atividade

Este repositório contém a resolução de quatro exercícios em JavaScript com foco no uso de arrays e operações manuais de manipulação de dados. Os programas foram desenvolvidos sem utilizar funções prontas de arrays para garantir o aprendizado das estruturas de vetores e dos algoritmos básicos.

## 📄 Objetivos

- Criar programas em `JavaScript` com extensão `.js`
- Testar cada programa para garantir funcionamento correto
- Utilizar vetores para aprimorar os algoritmos
- Manter versionamento local e remoto dos arquivos com `git`

## ➕ Atividades

1) Crie um programa que lê 10 números inteiros, armazena em um vetor e exibe a soma apenas dos números pares.
2) Criar um programa que preenche automaticamente um vetor com 7 números aleatórios e exibe os valores gerados.
3) Crie um programa que solicite 10 números inteiros, armazena em um vetor, solicita um número ao usuário e conta quantas vezes ele aparece no vetor.
4) Pedir ao usuário para digitar 10 números, armazena em um vetor, cria um novo vetor sem valores duplicados e exibe ambos.
